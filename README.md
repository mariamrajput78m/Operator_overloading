# 📚 Reading Tracker (C++)

A simple C++ program that demonstrates **operator overloading (`+=`)** by tracking and combining the number of pages read and days spent reading.

## 🛠️ Concepts Used

- C++
- Classes & Objects
- Constructors
- Operator Overloading (`+=`)
- Member Functions

## ▶️ Run

Compile and run the program using any C++ compiler.
